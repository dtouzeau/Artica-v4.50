[H5]Artica Gerir postfix e addons[/H5]
Se utilizar pacotes [b]reposit%C3%B3rios para debian ou RedHat (dpkg ou rpm)[/b]:[br]:[br
Tem 5 m%C3%B3dulos que pode descarregar na sourceforge.

[ul]
[li][b]artica-postfix-smtp-relay[/b][br]
Este pacote converte o artica para ser um SMTP Relay utilizando o programa principal postfix,ferramenta anti-spam bogofilter e saslauthd
[li][b]artica-postfix-security[/b][br]
Este pacote converte artica para um servidor de segurança SMTP completo usando Kaspersky Anti-spam e Kaspersky Anti-virus (licença com 30 dias para os avaliar).
[li][b]artica-postfix-mailbox[/b][br]
Este pacote converte artica para ser um servidor IMAP/POP3 de caixa de correio usando cyrus-imap
[li][b]artica-postfix-webmail[/b][br]
Este pacote adiciona um m%C3%B3dulo de webmail em artica usando RoundCube eMail e Mysql.
[/ul].

Existe uma ordem para instalar estes pacotes, por exemplo, se quiser instalar um servidor de correio completo (caixa de correio e webmail) deve instalar neste sentido : [br]
artica-postfix-smtp-relay -&amp;amp;raquo; artica-postfix-security -&amp;amp;raquo; artica-postfix-mailbox -&amp;amp;raquo;artica-postfix-webmail[br]
